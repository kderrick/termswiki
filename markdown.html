A lightweight markup language with plain text formatting syntax designed so that it can be converted to HTML. Markdown is often used to format readme files and create rich text using a plain text editor.

Example

On the left is text written in markdown. The right is how it will be rendered in the brower. Use the # symbol to create headers of different sizes (### is h3), * will create a list item, and more. Check out Dillinger to see more examples and see your markdown rendered in real time.
